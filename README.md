## 🚀 Overview

The **FD Calculator** is an interactive financial tool that helps users estimate their **Fixed Deposit (FD) returns** based on investment amount, interest rate, and time period.

It provides real-time calculations and a clean interface, making financial planning quick and easy.

---

## ✨ Features

- 💰 Adjustable investment amount  
- 📈 Interest rate control  
- ⏳ Time period selection  
- ⚡ Instant calculation of returns  
- 📊 Clear breakdown (Investment, Returns, Total Value)  
- 🎯 Clean and responsive UI
